Coze 插件推薦：台灣常用工具整合

CozeTW 團隊

2023年10月10日 · 閱讀時間：7分鐘

Coze 的強大之處在於其豐富的插件系統，透過整合各種實用工具，能讓你的 AI 助理更貼近台灣在地需求。本文精選了幾款台灣用戶最常用的 Coze 插件，並說明如何安裝與使用，幫助你打造更實用的自動化工作流。

1. 台灣氣象局天氣插件

這款插件整合了中央氣象局的數據，能提供台灣各地精確的天氣預報，包括溫度、降雨機率、紫外線指標等重要資訊。

安裝步驟：

  1. 進入 Coze 平台，點擊左側「插件」選項
  2. 搜索「台灣氣象局」或「Taiwan Weather Bureau」
  3. 點擊「安裝」按鈕，無需額外 API 金鑰

使用場景：

  • 每日定時發送台北市天氣提醒
  • 雨天自動觸發帶傘提醒
  • 依據紫外線指標建議防曬措施

範例程式碼片段：

{
  "action": {
    "name": "get_weather",
    "parameters": {
      "location": "台北市",
      "days": 3,
      "include": ["temperature", "rain_probability", "uv_index"]
    }
  }
}

2. 台灣新聞彙集插件

這款插件能自動收集台灣各大新聞媒體的頭條新聞，包括聯合新聞網、中國時報、自由時報等，並可依據類別過濾（政治、經濟、體育等）。

特色功能：

  • 支援自訂新聞來源偏好
  • 可設定關鍵字過濾，只收集感興趣的新聞
  • 提供新聞摘要功能，節省閱讀時間
  • 支持繁體中文顯示

3. 台灣地址解析與地圖插件

針對台灣地址格式優化的地圖插件，能正確解析縣市、鄉鎮、路名等地址資訊，並提供路線規劃功能，非常適合商業應用。

實用場景：

  • 餐廳訂位後自動發送地址與交通路線
  • 根據客戶地址規劃最有效率的外送路線
  • 活動通知時附上詳細地圖與交通方式

4. 台灣時鐘與行事曆插件

這款插件特別針對台灣時區和節假日進行了優化，能正確顯示台灣的國定假日、補班日等重要日期，是製作提醒類工作流的必備工具。

主要功能：

  • 自動辨識台灣國定假日
  • 支持農曆日期查詢
  • 可設定重複性事件（如每周會議）
  • 提前提醒功能，避免錯過重要日程

5. LINE 通知插件

台灣最受歡迎的通訊軟體非 LINE 莫屬，這款插件能讓 Coze 工作流直接發送訊息到你的 LINE 帳號或群組，即時接收重要通知。

設置步驟：

  1. 安裝 LINE Notify 應用程式
  2. 在 LINE Notify 中創建新的通知權杖
  3. 在 Coze 插件設定中輸入權杖
  4. 選擇要接收通知的對話或群組

插件組合建議

將多個插件組合使用能創造出更強大的功能：

  • 日常提醒組合：台灣氣象局插件 + LINE 通知插件
  • 商業資訊組合：台灣新聞插件（設定財經類） + Google 試算表插件
  • 活動安排組合：台灣時鐘插件 + 台灣地圖插件 + 郵件插件
