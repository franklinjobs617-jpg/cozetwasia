餐廳如何用 Coze 打造 LINE 訂位機器人

陳明軒 (餐飲科技顧問)

2023年10月28日 · 閱讀時間：12分鐘

在台灣餐飲業人力成本日益高漲的環境下，自動化訂位系統已成為餐廳提升效率的重要工具。本文將教你如何利用 Coze 平台，不需編碼知識也能打造一個功能完整的 LINE 訂位機器人，幫助餐廳節省人力、減少失約率。

你將學到：

  • 如何將 Coze 機器人與 LINE 連接
  • 設置訂位表單收集顧客資訊
  • 自動發送訂位確認與提醒訊息
  • 整合日曆管理訂位時段

為什麼選擇 LINE 作為訂位管道？

根據統計，台灣 LINE 活躍用戶超過2100萬，幾乎涵蓋所有年齡層。對餐廳而言，使用 LINE 作為訂位管道有以下優勢：

  • 用戶不需要安裝額外APP，門檻低
  • 可直接發送提醒訊息，打開率高達90%以上
  • 可累積顧客名單，便於後續行銷
  • 與台灣人的溝通習慣相符

步驟1：準備工作

開始之前，請確保你已擁有以下帳號與資源：

  1. Coze 平台帳號（免費註冊）
  2. LINE 企業帳號（可在 LINE Business Center 申請）
  3. 餐廳營業時間與可訂位時段表

步驟2：建立 LINE 機器人並連接 Coze

1. 進入 LINE Developer Console，創建新的 Messaging API 頻道

2. 在頻道設定中，找到「Channel Access Token」並複製

3. 打開 Coze 平台，進入「整合」頁面，選擇 LINE 整合

4. 粘貼剛才複製的 Access Token，完成連接設置

步驟3：設計訂位流程工作流

在 Coze Workflow 中，我們需要建立以下流程：

  1. 觸發條件：用戶發送「訂位」或點擊菜單按鈕
  2. 收集資訊：透過表單詢問用戶
    • 訂位日期與時間
    • 人數
    • 聯絡電話
    • 特殊需求（如素食、過敏原等）
  3. 可用性檢查：比對預訂時間是否有空位
  4. 確認訂位：發送確認訊息給用戶
  5. 記錄管理：將訂位資訊保存到 Google 試算表
  6. 提醒設置：預約前1天和1小時發送提醒

步驟4：設置自動提醒功能

減少顧客失約是餐廳訂位系統的重要功能。在 Coze 中設置雙重提醒：

  • 預約前1天：發送「明日訂位提醒」，包含時間地點
  • 預約前1小時：發送「即將到達提醒」，可附帶交通資訊

設置方法：使用「延時執行」組件，設定相對時間觸發提醒動作。

步驟5：導入顧客使用與後續優化

完成設置後，可將 LINE 機器人 QR 碼張貼在餐廳門口、菜單或宣傳物料上。建議：

  • 初期安排員工監控訂位狀況，確保系統運作正常
  • 收集顧客反饋，優化訂位流程
  • 節假日可增加「等候清單」功能
  • 定期分析訂位數據，優化營業策略

實際案例：台北某義大利餐廳的應用效果

位於台北東區的一家義大利餐廳導入此系統後，取得了顯著成效：

  • 訂位處理時間減少75%，節省一名服務生的人力成本
  • 失約率從20%降至8%
  • 顧客滿意度提升30%
  • 透過機器人累積了3000+會員名單
